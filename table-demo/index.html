Table Demo

A table us a way of creating a structured grid of data. The paired tages we will be insterested in are:

  • <table> : identifies the start of the table
  • <thead> : identifies the header row of the table
  • <tbody> : identifies the data detail body of the table
  • <tfoot> : identifies the footer row of the table
  • <tr> : identifies the start of the row on the table
  • <th> : identifies the cell on the header row for the column title
  • <td> : identifies the cell on the detail row or the footer row to hold content

Some tags have parameters. The border parameter of the table tag applies a border around your table. The colspan parameter and rowspan parameter are used on the <tf> tab to allow you to merge columns on a single row or column cells across rows.
